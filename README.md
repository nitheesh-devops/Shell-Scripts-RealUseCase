# Shell Scripts - Real Use Case 🚀

This repository contains **useful shell scripts** that are commonly used in **Linux server administration and DevOps environments**.  

The goal of this repository is to practice and demonstrate **automation tasks using Bash scripting**, which helps reduce manual work and improve operational efficiency.

---

# Why Shell Scripting?

Shell scripting is an essential skill for **DevOps Engineers and System Administrators** because it helps to:

- Automate repetitive tasks
- Monitor server health
- Manage files and directories
- Perform backups
- Handle log maintenance
- Improve operational efficiency

Most **Linux-based infrastructure automation** begins with simple shell scripts.

# Scripts Included

Below are the scripts available in this repository.

## 1️⃣ Check if File Exists
Checks whether a specific file exists in the system.

Useful for:
- Validation before running operations
- Deployment scripts

---

## 2️⃣ Check if Directory Exists
Verifies whether a directory exists.

Useful for:
- Backup scripts
- Log management

---

## 3️⃣ Disk Usage Alert Script
Checks disk usage and alerts if disk usage exceeds a defined threshold.

Useful for:
- Server monitoring
- Preventing disk full issues

---

## 4️⃣ Check if Service is Running
Checks whether a specific service or process is running.

Useful for:
- Monitoring critical services
- Health check scripts

---

## 5️⃣ Delete Log Files Older Than 7 Days
Automatically deletes log files older than 7 days.

Useful for:
- Log rotation
- Disk space management

---

## 6️⃣ Backup Directory with Date
Creates a backup of a directory and appends the current date to the backup folder.

Useful for:
- Backup automation
- Disaster recovery preparation

---

## 7️⃣ Fibonacci Series
Generates Fibonacci numbers up to a given limit.

Used for:
- Logic building
- Understanding loops in shell scripting

Example output:
```
0 1 1 2 3 5 8 13
```

---

## 8️⃣ Reverse a String
Reverses a given string input.

Example:
```
Input: DevOps
Output: spOveD
```

---

## 9️⃣ Factorial of a Number
Calculates the factorial of a given number.

Example:
```
5! = 120
```
